# javascript-meta

TS/JS files do not run natively!
Need a runner like Deno or NodeJs to run these files

if using Deno be sure to use;

    deno initialize workspace

run files with;

    npm start <name>
    OR
    node <name>
    OR
    ts-node <name>
    OR
    deno run <name>

run tests with

    npm test <name>
    OR
    deno test <name>

name is optional!
